● # Calculator Macronutrienți

  O aplicație web simplă care calculează necesarul zilnic de calorii, proteine, carbohidrați, grăsimi și hidratare, pe baza datelor personale introduse.

  ## Funcționalități

  - Calcul BMR folosind formula Mifflin-St Jeor
  - Ajustare în funcție de nivelul de activitate (TDEE)
  - Suport pentru 3 obiective: slăbire, menținere, masă musculară
  - Recomandare de hidratare personalizată
  - Descărcare rezultate în format `.txt`

  ## Cum se folosește

  1. Deschide `index.html` în browser
  2. Completează datele (sex, vârstă, greutate, înălțime, activitate, obiectiv)
  3. Apasă **Calculează**
  4. Opțional: descarcă rezultatele cu butonul **Descarcă rezultatele (.txt)**

  ## Tehnologii folosite

  - HTML5
  - CSS3 (responsive, inclusiv ecrane mici)
  - JavaScript (vanilla)
